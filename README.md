
## Getting Started

First, run the DB by entering 
'./pocketbase serve'
Use the Admin Ui provided in order to see the DB

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
